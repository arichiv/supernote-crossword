# supernote-crossword
Downloads tomorrow's crosswords into a designated Dropbox folder.

## Instructions:
1. Fork
2. Set DROPBOX_ACCESS_TOKEN, DROPBOX_NYTC_PATH, DROPBOX_WSJC_PATH, NYT_COOKIE secrets
3. Profit!
